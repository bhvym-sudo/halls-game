# halls-game
I made this game for SIH hackathon held at my college.
# how to play?
So this game is basically inspired by the japanese games such as exit-8 and shinkansen , where you have to escape the situations
lets talk about exit 8 , you have to basically spot the anomalies on each exit and if you are correct everytime you will reach exit 1 and game ends.
my game is setup in hostel building where the player is on 5th floor and he needs to spot anomalies on each floor to reach ground floor, I used lift for travelling on floors.
if there is any anomaly in any floor, and the player recognize the anomaly he will go upwards on the lift, to reach the next floor

# BUGS and changes

# Programming and frameworks
I used ursina engine of python to make this game, ursina is basically modified version of basic library in python called panda-3d, to make things easier to code
the developers developed ursina.
